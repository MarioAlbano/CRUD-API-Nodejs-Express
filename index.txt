//Node
-Executa Javascript fora do navegador
-V8 Engine
-Possui um núcleo
-Suporta várias operações simultaneamente (non-blocking)
-Podemos criar nossos própios módulos
-Já vem com módulos pré instalados
-Podemos instalar módulos com o npm

//Usando classes
-Criar classe, construtor com os this e os métodos, dar modules.exports
-No outro arquivo usar require e criar uma classe instanciada

//Iniciando
-npm init para começar o projeto
-Entrar no mongoDB cloud e configurar. Guardar login, password e link
-npm install express mongodb mongoose. Express = Framework popular para Node. MongoDB = Banco de dados não relacional. Mongoose = Biblioteca de modelagem de dados em objetos
